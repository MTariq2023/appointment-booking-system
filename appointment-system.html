Overview

This desktop application allows users to schedule and manage medical appointments. Built with C# and Windows Forms, it features PostgreSQL integration via Npgsql, real-time appointment listing, and input validation. Users can view scheduled appointments in a DataGridView and save new ones securely into a hosted PostgreSQL database.

Tech Stack: C#, .NET, Windows Forms, PostgreSQL, Npgsql, Visual Studio
